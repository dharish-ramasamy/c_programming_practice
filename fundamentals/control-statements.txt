[3] Jump Statements:

i). break:

break;

Used to exit:
	- loop
	- switch

ii). continue:

continue;

Use:
	- Skips current iteration of loop.

iii). goto:

goto label;

label:
    // statements

(Not recommended in good programming practice)

iv). return:

return value;

- Used to exit a function.
